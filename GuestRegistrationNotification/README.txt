The Guest Registration Notification mod displays a red, bordered box in the index as default. There are also options to display these boxes in the board index or above posts. Configure the mod through the admin control panel. This mod is compatible with MyBB 1.6.

• Editable content/title/display image
• Supports MyCode
• Utilizes hooks
